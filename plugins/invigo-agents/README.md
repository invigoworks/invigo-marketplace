# Invigo Agents

A collection of 23 specialized AI agents for Claude Code, designed to enhance your software development workflow.

## Included Agents

| Agent | Description |
|-------|-------------|
| `ai-engineer` | LLM application and RAG system specialist |
| `api-documenter` | OpenAPI/Swagger specs and developer documentation |
| `architect-review` | Code review for architectural consistency |
| `backend-architect` | Backend system architecture and API design |
| `code-reviewer` | Expert code review for quality and security |
| `context-manager` | Context management for multi-agent workflows |
| `database-architect` | Database architecture and design |
| `debugger` | Debugging errors and unexpected behavior |
| `deployment-engineer` | CI/CD and deployment automation |
| `devops-engineer` | DevOps and infrastructure |
| `error-detective` | Log analysis and error pattern detection |
| `frontend-developer` | React applications and responsive design |
| `fullstack-developer` | End-to-end application development |
| `javascript-pro` | Modern JavaScript with ES6+ |
| `mcp-expert` | Model Context Protocol integration |
| `mobile-developer` | Cross-platform mobile development |
| `prompt-engineer` | Expert prompt optimization for LLMs |
| `python-pro` | Idiomatic Python with advanced features |
| `search-specialist` | Expert web researcher |
| `task-decomposition-expert` | Complex goal breakdown |
| `test-engineer` | Test automation and quality assurance |
| `typescript-pro` | Advanced TypeScript type system |
| `ui-ux-designer` | UI/UX design specialist |

## Skills

- `skill-creator` - Guide for creating effective skills

## Installation

```bash
/plugin install invigoworks/invigo-marketplace invigo-agents
```

## Usage

Agents are automatically available through Claude Code's Task tool. Use them by specifying the appropriate `subagent_type`.
